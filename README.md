# Nour Downloader (Andorid/PC) 🚀

تطبيق تحميل الفيديوهات من يوتيوب، تيك توك، فيسبوك، وإنستجرام بضغطة واحدة!

### المميزات:
- تحويل الفيديو لـ **MP3** (صوت فقط).
- تحميل الفيديو بأعلى جودة **MP4**.
- واجهة المستخدم **Premium Dark Mode** بلمسات ذهبية شيك جداً.
- تحليل تلقائي للرابط وعرض صورة الفيديو وعنوانه قبل التحميل.

---

### المتطلبات (Requirements):
عشان تشغل البرنامج ده عندك، محتاج تثبت الحاجات دي:

1. **Python (installed):**
   ```bash
   pip install flet yt-dlp
   ```

2. **FFmpeg (مهم جداً للـ MP3):**
   - لازم تنزل `ffmpeg` وتخليه في الـ PATH بتاع الويندوز عشان يوتيوب يقدر يحول الصوت لـ MP3.

---

### طريقة التشغيل (Run):
افتح المجلد في الـ Terminal واكتب:
```bash
python main.py
```

### تحويل لـ Android (Build APK):
تقدر تستخدم أداة `flet build apk` عشان تطلع ملف APK للأندرويد مباشرة من الكود ده.
